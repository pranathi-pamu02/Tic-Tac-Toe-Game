# 🎮 Tic-Tac-Toe Web Application

A sleek, interactive, and responsive **Tic-Tac-Toe** game built with HTML, CSS, and JavaScript. This project was developed as part of the **SkillCraft Technology** internship/task.

## 🚀 Features

* **Two Game Modes:**
* **Player vs Player:** Compete against a friend locally.
* **Player vs Computer:** Test your skills against an automated opponent.


* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **Interactive UI:** Smooth transitions, glassmorphism effects, and dynamic status updates.
* **Win/Draw Detection:** Automatically detects winning combinations and highlights the winning path.

## 🛠️ Tech Stack

* **HTML5:** Structure and semantic layout.
* **CSS3:** Styling with Flexbox, CSS Grid, and custom animations.
* **JavaScript (ES6):** Game logic, state management, and DOM manipulation.

## 📂 Project Structure

```text
├── index.html   # Main structure and UI
├── style.css    # Styling and animations
└── script.js   # Game logic and AI moves

```

## 🖥️ How to Run

1. **Clone the repository:**
```bash
git clone https://github.com/pranathi-pamu02/Tic-Tac-Toe-Game.git

```


2. **Navigate to the project folder:**
```bash
cd Tic-Tac-Toe-Game

```


3. **Open the game:**
Simply open the `index.html` file in any modern web browser.

## 🕹️ How to Play

1. Select your game mode (2 Players or Vs Computer).
2. Click on any empty cell to place your mark (**X** or **O**).
3. The first player to align three marks horizontally, vertically, or diagonally wins!
4. Use the **Restart Game** button to reset the board at any time.
